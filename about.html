About the Historical Components of the DJIA Viz

Layout and any visual structure

This visualization follows a Martini Glass format. The narrow, author-led section starts with a brief overview of the changes the dow has experienced over the years. I picked these specific dates because I think it would help the user understand the sorts of changes that happen and why. The pictures I included for color and appearance. I found this easier to implement than CSS and I think it helps give the user a sense of control by being able to “go off the path” by spending some time on the pictures. The narrative then opens up and the user can then decide to see which changes occurred during whichever time period they are interested in.

Annotations

Tooltips are used to display the information I thought was most important about each specific company, which was name of the company and the date it was added. These appear when the user hovers over each bar and then they are cleared once the mouse is no longer hovering.

Parameters

There are two parameters - year and company. The visualization changes based on the desired display years. For example, if you inly want to view from 1910 to 2000, then you can put those dates in. This will interact with the companies displayed, such as the length of the bar chart (time it spend on the dow before being removed).

Triggers

The primary trigger of the visualization is inputting the date ranges. This is then linked to which companies are displayed on the chart. How the companies are displayed is determined by how many period they are active in the date range. For instance, if a company was added and removed twice, then there would be two different bars to indicate this.

User interface events

The user interface is primarily driven by buttons with labels directly on them containing directions (ie “Next”, “Explore”). When the narrative opens up, there are additional labels at the bottom to indicate the years can be adjusted. I decided to leave no affordance with regard to the tooltip because I think it is natural to hover over what you’re currently focused on, and additional text or markings would be distracting.
